# mini-parallax
Parallax library

```js
import MiniParallax from 'mini-parallax'

new MiniParallax('.js-parallax', {
  speed: 0.1
})
```

```js
import MiniParallax from 'mini-parallax'

new MiniParallax('.js-parallax', {
  isBackground: true
})
```
